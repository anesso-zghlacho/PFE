This project focuses on the development of a modular and real-time Network Intrusion Detection System (NIDS). With the increasing complexity of cyber threats, there is a growing need for efficient monitoring tools that can detect and respond to suspicious activities in real-time. Our solution integrates a high-performance packet capture engine using \texttt{libpcap}, a centralized analysis module, and a modern web-based dashboard for live traffic monitoring and threat visualization. The system is designed to provide security administrators with an intuitive interface to manage the capture process and analyze network flows, ensuring the integrity and security of the network infrastructure.

\textbf{Keywords:} Cybersecurity, NIDS, Packet Capture, Real-time Monitoring, Intrusion Detection.
